
## Installation

1. Clone the repo `git clone https://github.com/FlipZ3ro/INIT-Daily.git`

2. cp env.example .env

3. nano .env

3. Install the pkg `npm install`

## Start

Type `npm run start`

## Tasks Order

### Swapping

1. **INIT <> TOKEN:**

- INIT <> USDC
- INIT <> TIA
- INIT <> ETH
- INIT <> TUC

2. **TOKEN <> INIT**

- USDC <> INIT
- TIA <> INIT
- ETH <> INIT
- TUC <> INIT

### Staking

- DELEGATE INIT
- INIT <> USDC
- INIT <> ETH
- INIT <> TIA
