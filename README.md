## Prepare NODEJS V22

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
source ~/.bashrc
nvm install 22
nvm use 22
```

## Installation

1. Clone the repo `git clone https://github.com/FlipZ3ro/INIT-Daily.git`

2. `cp env.example .env`

3. `nano .env`

3. Install the pkg `npm install`

## Start

Type `npm run start`
